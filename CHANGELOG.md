# Change Log

All notable changes to the "lingpick-translation-extension" extension will be documented in this file.

## [0.1.0] - 2026-01-27

### Added
- Initial release of Lingpick Translation Extension
- Command to add translation keys to multiple locale files simultaneously
- Automatic detection of locale files in common directories (locales, i18n, lang, translations)
- Support for nested translation keys (e.g., "common.welcome", "button.submit")
- Interactive UI to input translation values for each locale
- Pretty-formatted JSON output with proper indentation
- Context menu integration for JSON files in the explorer
